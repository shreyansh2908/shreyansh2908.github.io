My Latest Projects

Explore More
Text Entailment Recognition

The project focuses on the development of a Natural Language Inference system designed to assess text entailment. The core challenge addressed by this project is the recognition of whether a given hypothesis can logically be inferred from a provided text fragment. This involves deep semantic analysis to understand and predict entailment, contradiction, and neutrality between pairs of text.

Image Annotator

It enhance image labeling. Click on an object, choose a predefined class or enter a custom comment. Supports uploading single files or entire folders. Features include undo and redo options for efficient workflow adjustments. Ideal for streamlining the annotation process in image-based projects.

Augmented Reality Watch Try-On App

The app allows users to virtually try on realistic watch models using their wrists. Built with C-Sharp, Unity, and Vuforia, this innovative application blends advanced AR technology to provide a unique and interactive shopping experience, enhancing consumer engagement and decision-making in real-time.

Enhanced XV-6 OS

Enhancements were made to the XV-6 operating system to include a system call trace feature, which intercepts and records system calls. Additionally, various scheduling policies were implemented. This project, utilizing C and GNU Make, aims to improve OS efficiency and reliability, providing deeper insights into system operations and resource management.
